Chartit
=======

This is for someone who need a CHART from CSV file.

Usage
=====


	Usage of ./chartit:
	  -csv string
	        CSV filename (default "input.csv")
	  -height int
	        OUTPUT file height (default 800)
	  -output string
	        OUTPUT filename (default "output.svg")
	  -width int
	        OUTPUT file width (default 1000)
